GUIDE

  1. Make sure your device meets the system requirements

    2. There are a few prerequisites to installing Windows 11, as the operating system will need more resources. Here are those system requirements:

    • A 64-bit, 1 GHz processor with at least two cores
    • 4 GB of RAM
    • 64 GB of storage
    • Trusted Platform Module (TPM), which ehances your device's security
    • UEFI firmware, which also improves your device's security
    • A graphics card with a WDDM 2.0 driver that is compatible with DirectX 12
    • An internet connection
  2. Go to the start menu

    3. If your system meets the specified requiremnets, we can begin installing windows. To start, select the start menu icon.

  3. Go to setting

    4. From the start menu, click on the gear icon above the power icon.

  4. Go to Update & Security

    5. Now that we're in the settings app, we want to click on "Update & Security".

  5. Check for updates

    6. Click on "Check for updates" and wait a few moments while your computer checks if Windows 11 is available to you.

  6. Download and Install

    7. If Windows 11 is available to you, the following box will pop up. Click on the "Download and Install" button.

  7. Accept Software License Terms

    8. Before installation, we'll need to accept the Software License Terms by clicking "Accept and Install" at the bottom of the window.

  8. Wait and Restart

    9. After you've accepted the Software License Terms, Windows 11 will begin installing. It may take between 20 minutes to an hour, but when it's done, you'll want to restart your computer by clicking the "Restart now" button.

Congratulations! You've just upgraded to Windows 11!
